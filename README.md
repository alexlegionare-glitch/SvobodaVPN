# Свобода VPN

Простой VPN-клиент для Windows для обхода блокировок и ТСПУ. Всё в одном окне: движок [sing-box](https://github.com/SagerNet/sing-box) (TUN на весь ПК) + VK-туннель. Поддержка VLESS / Reality / Hysteria2 / TUIC / Trojan / Shadowsocks / VMess. Один клик — и весь трафик ПК идёт через ваш сервер.

> ⚠️ Для личного использования. Своего сервера в комплекте нет — вы поднимаете его сами за 1 команду (см. ниже) или вставляете свою ссылку.

## Установка

1. Скачайте **[SvobodaVPN-Setup.exe](https://github.com/alexlegionare-glitch/SvobodaVPN/releases/latest/download/SvobodaVPN-Setup.exe)** из релизов.
2. Запустите его, подтвердите UAC («Да») — обычный мастер установки.
3. На рабочем столе появится ярлык «Свобода VPN»; VPN будет запускаться сам при входе в Windows.
4. Откройте ярлык, нажмите **«+ Добавить сервер»**, вставьте свою ссылку (`vless://`, `hysteria2://`, …) и нажмите большую кнопку.

(Есть и портативный вариант — [SvobodaVPN-Setup.zip](https://github.com/alexlegionare-glitch/SvobodaVPN/releases/latest/download/SvobodaVPN-Setup.zip): распаковать и запустить `Установить.bat`.)

Два режима (переключатель внизу окна):
- **Обычный** — кнопка вкл/выкл + «Добавить сервер». Для новичка.
- **Продвинутый** — «Серверы» (все настройки: протокол, транспорт, uTLS-отпечаток, Reality, mux) + «Подписка» с авто-обновлением списка.

## Свой сервер с нуля (для новичка)

Без знаний и копания в настройках. ~10 минут, ~150–300 ₽/мес.

1. Купите **VPS за рубежом** (ОС **Ubuntu 22/24**, доступ root). Дёшево: Hetzner, 4VPS, PQ.Hosting, Contabo. Запишите **IP** и **пароль root**.
2. Откройте веб-консоль сервера (Console / VNC) или подключитесь по SSH.
3. Вставьте одну команду:
   ```bash
   bash <(curl -fsSL https://raw.githubusercontent.com/alexlegionare-glitch/SvobodaVPN/main/setup.sh)
   ```
4. Через минуту скрипт напечатает **готовую ссылку** `vless://…` — скопируйте.
5. В приложении: **«+ Добавить сервер»** → вставьте ссылку → подключайтесь.

Скрипт сам ставит sing-box, делает сертификат и настраивает обход ТСПУ (**VLESS + TLS без SNI**).

## Если не подключается

1. **Нажмите кнопку ещё раз.** Блок ставится в момент соединения, и переподключение часто оживляет канал (приложение пробует это само).
2. **Смените сервер** или поднимите свой. Режут в основном по **IP/подсети сервера**, а не по протоколу — свежий чистый VPS обходит блок лучше любых настроек.
3. **Продвинутый режим → Серверы → uTLS-отпечаток:** переключите `chrome` ↔ `firefox`.

## Сборка из исходников

Бинарники (`sing-box.exe`, `wintun.dll`, `PWDTT.exe`) в репозиторий не входят — положите их рядом с `app.ps1` и запустите:
```powershell
powershell -ExecutionPolicy Bypass -File build-dist.ps1
```
Получите `dist/SvobodaVPN-Setup.zip`.
